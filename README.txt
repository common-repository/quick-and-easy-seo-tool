A light weight plugin that fixes all SEO loopholes in wordpress. Woocommerce Compatible.

== Description ==

Fixes all SEO loopholes and makes your blog 100% SEO friendly. Comes pre-configured, but is customizable. Woocommerce Compatible. Simple and easy to use. Yoast alternative.

= What does the plugin do? =

* Prevents duplicate pages and low value pages from getting indexed by search engines. 
* Adds 'Next' and 'Prev' tags to all archive pages & paginated single post pages (to indicate to Google that these pages are part of a sequence).
* Removes unwanted tags from wordpress head section (that helps increase page load speeds).
* Allows you to add custom title tags and meta description tags for homepage, posts, pages, media attachments, woocommerce and other post formats.
* Allows you to format the title tags.
* Adds canonical tags to all posts, pages and archives. 
* Optimizes woocommerce product pages, shop, category and tag pages.
* Adds important Facebook OG (Open Graph) tags - OG Title, OG Description, OG Image, OG Locale, OG Date Created and OG Date Modified.

If you need a new feature or have any doubts, feel free to contact me through my site: [https://orbitingweb.com/blog/quick-and-easy-seo-tool/](https://orbitingweb.com/blog/quick-and-easy-seo-tool/)


== Installation ==

= Manually installing the plugin =

1. Download the plugin zip file from Wordpress.org
1. Upload the zip file of the plugin by logging into your wordpress dashboard and going to 'Plugins > Add New > Upload Plugin'.
1. Click on the 'install' now button.
1. After the installation is complete, click the 'Activate' button.
1. Once activated, go to 'Settings > Quick And Easy SEO' to configure the plugin settings. The settings are already configured, so you can change them if need be.

= Installing from Wordpress dashboard =

1. Login to your wordpress dashboard and go to 'Plugins > Add New'.
1. Search for 'Quick And Easy SEO' and click on the 'Install' button.
1. Once the installation is complete, click on the 'Activate' button.
1. Once activated, go to 'Settings > Quick And Easy SEO' to configure the plugin settings. The settings are already configured, so you can change them if need be.
1. Check your HTML source to see if all tags are being added properly. 

== Frequently Asked Questions ==

= Can I use this along with Yoast or other SEO plugins? =
No, using this plugin with other SEO plugins will cause duplicate tags to be generated.

= Does the plugin fix all SEO related issues with WordPress? =
Yes, the plugin fixes all known SEO issues. It does not try to fix things that are already fixed and does not attempt to add functionalities that are an overkill and not required in most cases.

= Will the plugin slow down my site? =
Not in the least. This plugin only adds the bare minimum tags required to make your site SEO friendly and hence is very fast.

= Has the plugin been tested with multi-site? =
No, the plugin has not been tested with multi-site.

= What makes this plugin stand apart? =
The fact that it was built with simplicity in mind and is extremely simple to configure and manage.
